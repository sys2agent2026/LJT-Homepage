# Junteng Liu - Personal Academic Homepage

This is the personal academic homepage of **Junteng Liu**, built with the [academicpages](https://github.com/academicpages/academicpages.github.io) template.

## Quick Start

```bash
# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve
```

## Site Configuration

Edit `_config.yml` to update site metadata.
